
# Directorio: **data**

En este directorio guardaremos:
* **data/input**: Ficheros de entrada que se consumen al inicio de la ejecución.
* **data/output**: Ficheros de salida que se generan al final de la ejecución.
* **data/temp**: Ficheros temporales que puedan generar durante la ejecución.

> Recordar que el contenido de esta carpeta NO se "copia" en el servidor GitLab.
